Politique de Confidentialité

Dernière mise à jour : 11 décembre 2025

1. Introduction

Claque Doigts, édité par Line VON DACH, s'engage à protéger la confidentialité de vos données personnelles. Cette politique explique quelles données nous collectons, pourquoi nous les collectons, et comment nous les utilisons.

2. Données collectées

Nous collectons les données suivantes lorsque vous utilisez notre service :

  • Données d'identification : Prénom, nom, entreprise (optionnel)
  • Données de contact : Adresse email (optionnel), numéro de téléphone (optionnel)
  • Données de stratégie : Informations que vous renseignez dans le formulaire (cibles, plateformes, objectifs, parcours client)
  • Données techniques : Données stockées localement dans votre navigateur (localStorage) pour améliorer votre expérience

3. Finalité de la collecte

Vos données sont collectées pour les finalités suivantes :

  • Générer votre stratégie social media personnalisée
  • Créer un document PDF de votre stratégie
  • Sauvegarder votre progression dans le formulaire
  • Vous permettre de reprendre là où vous vous êtes arrêté
  • Vous contacter pour un rendez-vous (si vous le demandez)

4. Base légale du traitement

Le traitement de vos données repose sur :

Votre consentement : En utilisant notre service et en remplissant le formulaire, vous consentez au traitement de vos données personnelles.

L'exécution d'un service : Le traitement est nécessaire pour vous fournir le service de création de stratégie social media.

5. Stockage et conservation des données

Stockage local (localStorage) : Vos données sont stockées localement dans votre navigateur. Elles ne quittent jamais votre appareil sauf si vous choisissez la version payante.

Base de données Firebase (version payante uniquement) : Si vous utilisez la version complète avec authentification, vos données sont stockées de manière sécurisée sur Firebase (Google Cloud Platform) avec des règles de sécurité strictes.

Durée de conservation :
- Version gratuite : Les données restent dans votre navigateur jusqu'à ce que vous les supprimiez manuellement
- Version payante : Les données sont conservées tant que vous utilisez le service, avec possibilité de suppression à tout moment

6. Partage des données

Nous ne partageons, ne vendons, ni ne louons vos données personnelles à des tiers.

Les seuls accès à vos données sont :

Vous-même : Vous avez un accès complet à vos données
Line VON DACH : Uniquement pour la version payante, afin de vous fournir le service et le support
Firebase (Google) : En tant qu'hébergeur technique de la base de données, soumis à des obligations strictes de confidentialité

7. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : Vous pouvez demander une copie de toutes vos données
  • Droit de rectification : Vous pouvez corriger vos données inexactes
  • Droit à l'effacement : Vous pouvez demander la suppression de vos données
  • Droit à la portabilité : Vous pouvez récupérer vos données dans un format structuré
  • Droit d'opposition : Vous pouvez vous opposer au traitement de vos données
  • Droit de retirer votre consentement : À tout moment, sans affecter la licéité du traitement fondé sur le consentement effectué avant le retrait

Pour exercer ces droits, contactez-nous à : [email protected]

8. Sécurité

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données contre tout accès non autorisé, modification, divulgation ou destruction :

- Chiffrement des connexions (HTTPS)
- Règles de sécurité strictes sur Firebase
- Accès limité aux données (uniquement vous et l'administrateur)
- Stockage sécurisé sur des serveurs certifiés (Google Cloud Platform)

9. Cookies et technologies similaires

Ce site n'utilise pas de cookies de suivi ou publicitaires.

Nous utilisons uniquement le localStorage de votre navigateur pour :
- Sauvegarder votre progression dans le formulaire
- Mémoriser vos préférences
- Améliorer votre expérience utilisateur

Vous pouvez désactiver le localStorage dans les paramètres de votre navigateur, mais cela peut affecter le fonctionnement du site.

10. Modifications de la politique

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une date de mise à jour.

Nous vous encourageons à consulter régulièrement cette page pour prendre connaissance des éventuelles modifications.
